# personal-portfolio
Project Overview
This repository contains the source code for the personal portfolio of Rodrigo Molero, a Computer Engineering student at the University of South Florida. The project serves as a centralized platform to showcase academic research in robotics, technical implementations of low-level systems, and leadership within the IEEE Computer Society.

Technical Architecture
The website is a custom-built frontend application utilizing a clean, separation-of-concerns architecture across HTML, CSS, and JavaScript.

Core Stack
HTML5: Semantic structure for sections including Research, Experience, and Skills.

CSS3: Custom layout management featuring Flexbox, box-shadow depth effects, and smooth-scroll behavior.

JavaScript (ES6): Functional logic for theme management, scroll-based UI visibility, and client-side form processing.

Key Features
Dark Mode Toggle: A JavaScript-driven theme switcher that dynamically updates CSS classes to modify the visual environment from high-contrast light mode to a professional dark aesthetic.

Interactive Contact System: A client-side validated form that provides real-time user feedback upon submission.

Responsive Media Integration: Native HTML5 video embedding for ERA Lab simulation demonstrations and integrated PDF viewing for professional credentials.

Navigation Utilities: A "Back-to-Top" floating action button that appears dynamically after a 400px scroll threshold to enhance user experience on long-form content.

Project Portfolio Highlights
The repository documents several significant engineering projects:

ISA Assembler Compiler: A custom compiler for transforming Instruction Set Architecture (ISA) assembly into machine-executable code.

Student Debt Manager: A C-based financial tool for tracking balances, interest rates, and monthly payment schedules.

Robotics Simulation: Video documentation of underwater navigation algorithms developed within NVIDIA Isaac Sim during undergraduate research at the ERA Lab.

File Structure
index.html: The main entry point containing semantic sections for navigation, projects, and professional history.

style.css: Centralized styling including theme variables for .dark-mode and UI component layouts.

script.js: Event listeners and logic for dark mode, scroll behavior, and form handling.

resources/: Directory housing the ERA Lab simulation media and the professional resume PDF.

Deployment & Usage
Clone the repository to your local machine.

Ensure the resources/ directory is present to maintain media and document links.

Open index.html in a standard web browser to view the portfolio.
